1) Описание.
Клиенты ClientCPlus.exe и ClientCSharp.exe представляют собой простую систему передачи сообщений между собой. 
Для передачи сообщений, устанавливается точка подключения, после чего клиенты готовы к работе. 
Настройка точки подключения пыполняется в конфигурационном файле config.ini. 
Оба клиента могут находиться в одой папке, а могут в разных, включая папки на разных машинах.
При расположении клиентов в одной папке, требуется только один конфигурационный файл config.ini.
В случае разных папок, для каждого клиента требуется свой файл config.ini, при этом содержимое обоих конфигов должно быьт одинаково.

2) Настройка параметров точки подключения.
В файле config.ini, в разделе [Connection] установите IP адрес сервреа в парметре Server и порт в параметре Port
Так, в случае запуска клиентов на одной и той же машине, конфигурационый файл должен быть таким:

[Connection]
Server=192.168.1.110
Port=9990

здесь, IP сервера = 192.168.1.110, а порт = 9990
